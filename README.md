\# Python Pomodoro Timer Desktop App 🍅



A simple Pomodoro Timer desktop application built using Python.  

This project was originally created as a Python script and later improved, debugged, and packaged into a Windows desktop application.



\## 📌 About the Project



This is a mini productivity app based on the Pomodoro technique. It helps users focus by dividing time into work sessions and break sessions.



The app includes:

\- Work timer

\- Short break timer

\- Long break timer

\- Animated GIF support

\- Session tick marks

\- Reset functionality

\- Desktop `.exe` version using PyInstaller



\## 🛠️ Technologies Used



\- Python

\- Tkinter

\- Pillow

\- PyInstaller



\## ✨ Features



\- 25-minute work sessions

\- 5-minute short breaks

\- 20-minute long breaks

\- Animated GIFs for different timer modes

\- Visual tick marks to track completed work sessions

\- Reset button to restart the timer

\- Packaged as a Windows desktop application
